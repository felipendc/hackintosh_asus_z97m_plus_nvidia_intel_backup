***Quick Tip: How to mount the EFI folder from a Disk Driver or USB Stick using Linux:.***

> sudo fdisk -l
> Linux syntax: sudo mount /dev/PARTITION-ID /MOUNT-LOCAL
> Example: sudo mount /dev/sdd1 /media
